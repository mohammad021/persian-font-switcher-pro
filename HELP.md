# راهنمای Persian Font Switcher Pro

## امکانات اصلی

### فونت‌ها
- دسته‌بندی گوگل، راستکردار، سری B
- فونت پیش‌فرض
- افزودن فونت سفارشی

### Picker
- 🎯 **استثنا**: المان‌هایی که فونت اعمال نشود
- 🎨 **اختصاص فونت**: فونت خاص به بخش خاص

### تنظیمات per-site
- هر سایت فونت، استثنا و RTL جداگانه دارد

## استفاده
1. پنل را باز کنید (Ctrl+Shift+F)
2. فونت انتخاب کنید
3. برای picker از دکمه‌ها استفاده کنید

## مشکلات رایج
- اگر فونت لود نشد، کش را پاک کنید
- برای سایت‌های خاص از ویرایش سایت استفاده کنید
