# Persian Font Switcher Pro v12

**ابزار حرفه‌ای تغییر فونت فارسی برای مرورگر**

## امکانات کلیدی
- فونت‌های آماده گوگل، راستکردار، سری B
- افزودن فونت سفارشی (لینک یا فایل)
- انتخابگر المان (Element Picker) برای استثنا یا اختصاص فونت خاص
- تنظیمات جداگانه برای هر سایت
- پشتیبانی RTL/LTR
- پنل زیبا با تم تیره/روشن
- مینیمایز به دکمه شناور یا pinned
- حافظه کش فونت‌ها
- میانبر Ctrl+Shift+F
- به‌روزرسانی خودکار
